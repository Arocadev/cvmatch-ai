Groq API Token

CVMatch AI uses Groq to run the AI models that analyze your CV, summarize job offers, and generate adapted CVs. Groq offers a free API with a generous usage limit — no credit card required.

The app works out of the box with a shared token. If you add your own, you get dedicated usage limits.

1
Go to console.groq.com and create a free account.
2
Click API Keys in the left sidebar.
3
Click Create API Key, give it a name and confirm.
4
Copy the key — it starts with gsk_. You will only see it once.
5
Go to your Profile in CVMatch AI and paste it in the Groq token field.
Your token is encrypted with AES-256 before being stored. Never shown in plain text after saving.

Yes. The free plan includes 14,400 requests/day, no credit card required. A typical CVMatch AI session uses 3–5 requests.

  • If you added your own token, make sure it starts with gsk_ and was copied completely.
  • Verify the token is still active at console.groq.com.
  • If you hit rate limits, wait a few minutes and try again.

Job search

Job data comes from external APIs. Some international companies post in English regardless of country. Try Adzuna España as the source for more Spanish-language results.

  • New — Retrieved from the last search, not yet reviewed.
  • Viewed — Opened at least once.
  • Saved — Marked for follow-up.
  • Discarded — Can be permanently deleted from the Discarded tab.

CV and PDF

Yes, up to 3 CVs in your profile (PDF or text). When analyzing an offer, you can choose which one to use.

Automatic: uses the AI-generated CV adapted to the specific job offer.

Manual: you write or paste the CV content yourself in the PDF creator.

Token de API de Groq

CVMatch AI usa Groq para ejecutar los modelos de IA que analizan tu CV, resumen las ofertas y generan CVs adaptados. Groq ofrece una API gratuita con un límite de uso generoso — sin tarjeta de crédito.

La app funciona directamente con un token compartido. Si añades el tuyo, tendrás límites de uso dedicados.

1
Ve a console.groq.com y crea una cuenta gratuita.
2
Haz clic en API Keys en el menú lateral izquierdo.
3
Pulsa Create API Key, ponle un nombre y confirma.
4
Copia la clave — empieza por gsk_. Solo la verás una vez.
5
Ve a tu Perfil en CVMatch AI y pégala en el campo de token Groq.
Tu token se cifra con AES-256 antes de guardarse. Nunca se muestra en texto plano tras guardarlo.

Sí. El plan gratuito incluye 14.400 peticiones/día, sin tarjeta de crédito. Una sesión típica en CVMatch AI consume entre 3 y 5 peticiones.

  • Si has añadido tu propio token, comprueba que empiece por gsk_ y se haya copiado completo.
  • Verifica que el token sigue activo en console.groq.com.
  • Si has alcanzado el límite de peticiones, espera unos minutos e inténtalo de nuevo.

Búsqueda de ofertas

Los datos provienen de APIs externas. Algunas empresas publican en inglés independientemente del país. Prueba a usar Adzuna España como fuente para obtener más resultados en castellano.

  • Nueva — Obtenida en la última búsqueda, aún no revisada.
  • Vista — Abierta al menos una vez.
  • Guardada — Marcada para hacer seguimiento.
  • Descartada — Se puede eliminar definitivamente desde la pestaña de descartadas.

CV y PDF

Sí, hasta 3 CVs en tu perfil (en PDF o texto). Al analizar una oferta, puedes elegir cuál usar.

Automático: usa el CV generado por la IA, adaptado a la oferta concreta.

Manual: escribes o pegas el contenido del CV directamente en el creador de PDF.
