{% if idioma == 'en' %}How does it work?{% else %}¿Cómo funciona?{% endif %}

1

{% if idioma == 'en' %}Configure your search{% else %}Configura tu búsqueda{% endif %}

{% if idioma == 'en' %}Go to the search page and enter your keywords and location. You can also filter by modality and experience.{% else %}Ve al buscador y pon las palabras clave que buscas y la ciudad o zona. También puedes filtrar por modalidad y experiencia requerida.{% endif %}

2

{% if idioma == 'en' %}Review offers{% else %}Revisa las ofertas{% endif %}

{% if idioma == 'en' %}Offers appear in the New list. 3

{% if idioma == 'en' %}Organize your favorites{% else %}Organiza las que te interesan{% endif %}

{% if idioma == 'en' %}Save the ones you like, discard the ones you don't. Saved offers stay in your Saved list. Discarded ones can be deleted whenever you want to clean up.{% else %}Guarda las que te interesen, descarta las que no. Las que guardes quedan en tu lista de Guardadas. Las descartadas puedes eliminarlas cuando quieras limpiar.{% endif %}

4

{% if idioma == 'en' %}Analyze your CV with AI{% else %}Analiza tu CV con IA{% endif %}

{% if idioma == 'en' %}From any offer you can analyze your CV. {% if idioma == 'en' %}And also...{% else %}Y además...{% endif %}

{% if idioma == 'en' %}Your CV is remembered — once you upload your CV, it is remembered for the entire session. {% if idioma == 'en' %}Fresh searches — every time you search again, new offers are updated automatically.
